〈h5|打印〉

在元素之前插入：document.getElementById('【id】').insertAdjacentHTML('beforebegin',"内容");
在元素之后插入：document.getElementById('【id】').insertAdjacentHTML('afterend',"内容");

在元素内的结尾：document.getElementById('【id】').insertAdjacentHTML('beforeend',"内容");

〈h4|if 判断〉
a==ba是否等于b（值相等就行）
a===ba是否等于b（值和类型都要相等）
a!=ba是否不等于b（值不相等就行）
a!==ba是否不等于b（值和类型都要不相等）

>大于　<小于　>=大于或等于　<=小于或等于
&& 和　|| 或　! 不
(x<10 && y>10) x小于10并且y大于10

if(a<10){b="你好";}else{b="不好";}
b=(a<10)?"你好":"不好";
如果a小于10 则b等于"你好"，否则"不好"

if(条件1){则A}else if(条件2){则B}else{则C}
如果满足1则A，否则如何满足2则B，否则C

if(变量.indexOf("字符")!=-1){
如果变量包含"字符"
}

判断变量存在：if(typeof 变量!=="undefined")

〈h5|简化 if else〉
条件="a";if(条件=="a"){}else if(条件=="b"){}else{}
简化为↓
const a={
  "a":(参数)=>console.log("a"),
  "b":(参数)=>alert("b"),
}
if(a[条件]){
  a[条件](参数);
}else{
  console.log("c");
}

存在其他判断条件，简化为↓
const b=[
  [()=>条件.indexOf("a")!=-1,()=>console.log("a")],
  [()=>条件.length==2,()=>console.log("b")],
];
const c=b.find((m)=>m[0]());
if(c){
  c[1]();
}else{
  console.log("c");
}
〈h4|循环〉
break; 跳出循环
continue; 暂时跳过这次循环（比如每次循环+1，结果是1234，在第3次执行continue，12之后会执行4，变成124）

for(let 变量=0;变量<5;变量++){循环内容}
变量初始为0，每次循环+1，如果变量小于5就继续循环，否则停下

while(变量<5){循环内容}
只要变量小于5就会循环下去（小心不要写出无限循环）

do{循环内容}while(判断); （;一定要有） 先执行一次，再判断是否满足
