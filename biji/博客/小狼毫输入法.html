下载和安装
  1. 下载小狼毫输入法（rime.im）
  2. 下载雾凇词库（github.com/iDvel/rime-ice/releases/tag/nightly），解压放进“用户文件夹”
  3. 下载语法模型（github.com/amzxyz/RIME-LMDG/releases，文件名可能是“wanxiang-lts-zh-hans.gram”，约200mb），放进“用户文件夹”
  4. 创建配置文件“rime_ice.custom.yaml”（内容参考）
  5. 把备份的数据复制过来（目录/子/备份/小狼毫输入法）


〈打开|配置文件|html|C:/mq/app/huancun/xiaolanghao/default.yaml〉
〈打开|猫球词库|html|C:/mq/app/huancun/xiaolanghao/pvo.dict.yaml〉
〈h3|自定义短语〉
  1. 〈打开|编辑|html|C:/mq/目录/子/备份/小狼毫输入法/词库_复制.html〉
  2. 打开 → 全选 → 复制
  3. 〈打开|打开|html|C:/mq/app/huancun/xiaolanghao/custom_phrase.txt〉 → 全选 → 粘贴 → 保存
  4. 右键小狼毫图标 → 重新部署
