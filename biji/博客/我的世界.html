〈h4|药水酿造〉
将烈焰粉（作为燃料）、材料、水瓶放入酿造台。
通常的流程：
水瓶+下界疣=粗制的药水（有些药水不需要做成粗制的药水）
粗制的药水+B类材料=有效果的药水
有效果的药水+A类材料=加强版药水
例子：
水瓶+下界疣=粗制的药水
粗制的药水+糖=迅捷药水（持续3分钟）
迅捷药水+红石=迅捷药水（持续8分钟）

A类材料：
材料成品效果外形
下界疣粗制的药水无深蓝
萤石粉浓稠的药水提升效果等级深蓝
红石平凡的药水延长持续时间深蓝
发酵蛛眼虚弱药水腐化效果（如治疗变成伤害）紫色 发光
火药喷溅水瓶药水变为喷溅药水深蓝 投掷瓶
(需要喷溅水瓶)龙息滞留药水喷溅药水变为滞留药水深蓝 滞留瓶

B类材料：
材料成品效果外形
糖迅捷药水速度浅蓝 发光
兔子脚跳跃药水跳跃提升浅绿 发光
闪烁的西瓜片治疗药水瞬间治疗品红 发光
蜘蛛眼剧毒药水中毒绿色 发光
河豚水肺药水水下呼吸蓝色 发光
岩浆膏抗火药水防火橙色 发光
金胡萝卜夜视药水夜视蓝色 发光
烈焰粉力量药水力量深粉 发光
恶魂之泪再生药水生命恢复粉红 发光
海龟壳神龟药水抗性提升、缓慢紫色 发光
幻翼膜缓降药水缓降白色 发光
糖 + 发酵蛛眼迟缓药水缓慢淡紫 发光
兔子脚 + 发酵蛛眼迟缓药水缓慢淡紫 发光
闪烁的西瓜片 + 发酵蛛眼伤害药水瞬间伤害深紫 发光
蜘蛛眼 + 发酵蛛眼伤害药水瞬间伤害深紫 发光

〈h3|指令〉
更详细、完整的说明参考我的世界百科 链接
/gamemode creative 复制游戏模式变更为创造模式
/gamemode survival 复制游戏模式变更为生存模式
/give 复制给予玩家物品
/give @s minecraft:player_head{SkullOwner:""} 复制获得一个玩家的头

时间：〈复制|白天|/time set 1000〉、〈复制|中午|/time set 6000〉、〈复制|晚上|/time set 13000〉、〈复制|半夜|/time set 18000〉 | 〈复制|时间固定|/gamerule doDaylightCycle false〉、〈复制|时间流逝|/gamerule doDaylightCycle true〉
天气：〈复制|晴天|/weather clear〉、〈复制|雨天|/weather rain〉、〈复制|雷暴雨|/weather thunder〉

坐标：x轴 y轴 z轴 水平旋转 垂直旋转
其中x轴为东西方向，y轴为上下高度，z轴为南北方向。
输入数字是绝对坐标，比如/tp 1 2 3，是坐标为1 2 3的特定位置。
~ 波浪号为相对坐标，比如/tp ~ ~ ~3，是以玩家为中心向南移动3点（单独的波浪号~视为~0，即不改变）。
^ 为局部坐标，比如/tp ^ ^ ^3，是向玩家前方移动3点。
x=?,y=?,z=?,dx=?,dy=?,dz=? 以绝对坐标和相对坐标画一个长方体
distance=1..2 1和2之间
distance=..2 小于2
distance=1.. 大于1

目标选择器：
@p：距离最近的玩家。
@r：随机玩家。
@a：所有玩家。
@e：所有实体，包括玩家，但不包括死亡的实体。
@s：命令的执行者。
