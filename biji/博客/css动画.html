〈https|www.w3school.com.cn/cssref/pr_animation.asp|教程〉

animation:动作名 持续时间 次数(infinite无限);

或

animation-name:动作名;
animation-duration:时间;
animation-timing-function:速度曲线; linear匀速
animation-delay:开始前的延迟;
animation-iteration-count:次数;
animation-direction:是否轮流反向播放

@keyframes 动作名{
    0%{
        width:100px; 宽度 同理height
        color:#ffffff; 颜色 同理background-color

        transform: scale(1.0); 放大
        transform:rotate(0deg); 角度
        transform:rotateY(0deg); 翻转
        〈https|www.w3school.com.cn/cssref/pr_transform.asp|教程〉
        opacity: 1.0; 不透明度
    }
}
