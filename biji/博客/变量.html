〈h5|截取特定位数〉
变量=变量.slice(开始位置,开始位置+保留位数); 开始位置第一位是0，第二位是1，以此类推
比如 变量="123456789";
变量.slice(0,3); 123
变量.slice(1,1+3); 234

〈h5|删除特定位数〉
适用于字符串和数组
变量="12345"; 或["1","2","3","4","5"]
变量=变量.slice(3); 删除前3位 结果为："45"或["4","5"]
变量=变量.slice(-3); 删除后3位 结果为："12"或["1","2"]

从特定位置删除：变量.splice(开始位置,删除的位数);
直接修改变量 变量.splice(0,1)，而不是 变量=变量.xxx

〈h4|随机〉
随机数：Math.floor(Math.random()*值); 值是从几个数字里抽取，比如1就只会出现"0" let 抽取其一=数组[Math.floor(Math.random()*数组.length)];

function 随机打乱(arr){
    for(let i=arr.length-1;i>0;i--){
        const j=Math.floor(Math.random()*(i+1));
        [arr[i],arr[j]]=[arr[j],arr[i]];
    }
    return arr;
}
let 数组=["1","2","3","4"];
数组=随机打乱(数组);
