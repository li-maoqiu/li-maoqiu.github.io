〈h5|外边距〉〈复制|margin|margin〉 后面可以写1到4个数字
1：所有方位
1 2：上下 左右
1 2 3：上 左右 下
1 2 3 4：上 右 下 左
如果两个设置了外边距的元素贴在一起，不是两个值相加，而是只取最大的一个数字。

〈h4|选择器〉
*{所有}
.a{class="a"}
.a.b{class="a b"}
a,b{多个选择器}
a b{a里的b，之间可以有其他层}
a>b{在a里的b(<a><b>可以)，<a><C><b>这种不行}
a+b{在a后面紧邻的b}
a~b{在a后面所有的b}
#a{id是a}
[a]{有a标签}
[a="b"]{a标签的值是"b"}
[a^="b"]{a标签的值以"b"开头（通配符）}
[a$="b"]{a标签的值以"b"结尾}

p:first-letter 段落首字
p:first-line 段落首行
p:before 段落最前 after 段落最后
.zhushi:before{content:"//";}

:first-child 所有父元素里的第一个子元素
（p:first-child 第一个子元素且是p标签 如果是其他标签则不生效）
:last-child 最后一个子元素
:nth-child(x) 第x个子元素 (even)偶数 (odd)奇数

:first-of-type 所有父元素里的第一个同类子元素
last nth同理

:not(选择器) 排除

:has() 有
a:has(b) <a>里有<b>则生效
a:has(+b) <a>后面是<b>则生效
a:has(b[ceshi]) <a>里的<b>是<b ceshi>则生效

〈h5|鼠标〉
:link 未访问
:visited 已访问
:hover 悬浮
:active 活动

〈h5|优先级〉
内联样式 1000
id选择器 100
类和伪类 10
元素选择器 1
* 0
继承样式 无
包含多个选择器时，将优先级相加
不会超过最大数量级（比如10个类选择器不会是10x10=100）
优先级相同，使用靠后的样式
在样式最后加 !important 则它的优先级最高（不推荐使用）

〈h5|当内容溢出框时处理〉 overflow
overflow:visible 显示
overflow:hidden 溢出部分隐藏（可以解决高度塌陷问题）
overflow:auto 溢出部分隐藏，显示滚动条

〈h5|元素框类型〉display:
none 不显示
block 块元素
inline 内联元素
inline-block 行内块元素 既可以设置宽高 也不独占一行

〈h5|元素是否可见〉 visibility:
visible 可见
hidden 不可见 隐藏但是依然占位

〈h5|定位〉 position:
static 默认值 没有定位
absolute 绝对定位
fixed 固定定位 相对于浏览器窗口的绝对定位
relative 相对定位：相对于自身原本位置偏移，原本位置依然占位。提升一个层级

overflow: hidden;
text-overflow: ellipsis;
white-space: nowrap;

〈h5|清除元素所有样式〉
all:unset
